# Webページ作成における工夫点

以下の点に工夫を凝らしました。
URL: https://awlts.github.io/italian-restaurant/

## 1. デザインと雰囲気作り
*   **高級感のある配色**: ワインレッド（`#800020`）とゴールド（`#d4af37`）を基調とし、落ち着いた大人の隠れ家的な雰囲気を演出しました。
*   **視覚的なインパクト**: ファーストビュー（ヒーローセクション）に動画または高品質な画像を大きく配置し、訪問者の興味を惹きつけます。
*   **ロゴデザイン**: CSSのみでイタリア国旗カラー（緑・白・赤）のグラデーションアイコンを作成し、店名の「CIT」に彩りを添えました。
*   **アニメーション**: スクロールに合わせてコンテンツがふわっと表示されるフェードインアニメーションを採用し、優雅な印象を与えています。

## 2. ユーザー体験の向上
*   **レスポンシブ対応**: スマートフォンからPCまで、どのデバイスでも快適に閲覧できるよう、レイアウトが自動的に最適化されます。
*   **ハンバーガーメニュー**: モバイル表示時にはナビゲーションをハンバーガーメニューに収納し、画面を広く使えるようにしました。
*   **トップへ戻るボタン**: ページを読み進めた後、ワンクリックでスムーズに最上部へ戻れるボタンを右下に配置しました（スクロール量に応じて自動表示）。

---

## 3. 第２回課題での追加要件対応

### （１）GIFアニメーションの作成と使用
*   **ワインアニメーション画像**: AI画像生成を活用し、ワインをモチーフにした画像を作成しました。
*   **配置場所**: トップページ（`Home.jsx`）の「Welcome to CIT」セクションに配置し、お店の雰囲気を視覚的に伝えています。
*   **工夫点**: 高級感のあるワイングラスと温かみのあるレストランの雰囲気を表現し、訪問者に特別な体験を予感させるデザインとしました。

### （２）動画像や音声の追加
*   **動画**: 
    - トップページのヒーローセクションに背景動画（`ワイン.mp4`）を配置
    - メニューページに料理の様子を紹介する動画セクション（`パスタ.mp4`）を追加
*   **音声（BGM）**: 
    - **ヘッダーに音声コントロールを配置**（フッターから移動して、トップへ戻るボタンとの重複を解消）
    - ユーザーの操作で再生開始（自動再生は避け、UXに配慮）
    - 音量調整スライダーを実装し、ユーザーが好みの音量で楽しめるように配慮

### （３）Reactフレームワークの使用
*   **Vite + React 18**: 高速な開発環境とモダンなコンポーネントベースの設計
*   **React Router DOM v6**: クライアントサイドルーティングによるスムーズなページ遷移
*   **コンポーネント構成**:
    - `Header.jsx`: ナビゲーション + 音声コントロール（グラスモーフィズムデザイン）
    - `Footer.jsx`: フッター
    - `BackToTop.jsx`: トップへ戻るボタン（useState/useEffectによる表示制御）
    - `Home.jsx`, `Concept.jsx`, `Menu.jsx`, `Access.jsx`: 各ページコンポーネント
*   **モダンなスタイル**:
    - グラスモーフィズム効果（背景ぼかし + 半透明）
    - グラデーションアクセントカラー
    - カードホバーエフェクト
    - スムーズなトランジション

## 4. 技術的な配慮
*   **GitHub Pages対応**: `vite.config.js` で `base: '/italian-restaurant/'` を設定し、GitHub Pagesで正しく動作するようにしました。
*   **HashRouter使用**: GitHub Pagesでの直接URLアクセスに対応するため、BrowserRouterではなくHashRouterを使用しました。
*   **コンポーネントの再利用**: ヘッダー、フッター、トップへ戻るボタンを共通コンポーネント化し、保守性を向上させました。

## 5. 開発コマンド
```bash
npm install       # 依存関係のインストール
npm run dev       # 開発サーバー起動
npm run build     # 本番ビルド
npm run preview   # ビルド結果のプレビュー
```
